# Data Source

## Status

Dataset selection is in progress.

## Selection criteria

The selected retail dataset should include:

- Customers
- Products
- Orders
- Order items
- Payment information
- Timestamps
- Stable identifiers
- A clear usage license
- Enough rows for profiling
- A reproducible download method

## Data handling policy

Raw source data will not be committed to GitHub.

The repository will provide:

- Dataset source information
- Download instructions
- An automated download script
- File checksums where practical
- Small test fixtures where redistribution is permitted
